# 📊 نظام الحسابات الذكي للشركات | 

نظام محاسبي متكامل ومطور يهدف إلى إدارة شجرة الحسابات (دليل الحسابات) والقيود اليومية والتقارير المالية بشكل مرن وبواجهات احترافية ومستقرة. تم بناء النظام باستخدام تقنيات ميكروسوفت الحديثة مع التركيز على تجربة مستخدم (UX) متطورة تتفادى أخطاء الأنظمة المحاسبية التقليدية.

---

## 🚀 الميزات الرئيسية التي تم تنفيذها

### 1. دليل الحسابات الذكي (Smart Accounts Tree)
تم تطوير صفحة الشجرة بالكامل لتصبح تفاعلية واحترافية:
* **نظام التبعية الديناميكي (Parent Selector):** كمبو بوكس ذكي يوضح هيكل الحسابات ومستوياتها (Level)، ويسمح بنقل الحساب أو تغيير الحساب الأب برمجياً بسهولة دون التأثير على سلامة البيانات.
* **التوليد التلقائي للأكواد (Auto Code Generation):** يقوم النظام تلقائياً باحتساب الكود المحاسبي التالي للحساب الفرعي بناءً على كود الحساب الأب (مثال: الأب `11` -> الابن الأول `1101`).
* **التحكم التلقائي في خصائص الحساب:** * منع تحويل الحسابات الأب إلى حسابات حركة (Postable) إذا كانت تحتوي على حسابات فرعية لضمان سلامة الترحيل المحاسبي.
  * نظام أرشفة متطور (Archiving) يعطل الحساب بصرياً وفي العمليات دون حذفه نهائياً للحفاظ على تاريخ القيود.
  * منع الحذف النهائي للحسابات المرتبطة بحركات أو قيود يومية سابقة.

### 2. لوحة التحكم والرسوم البيانية (Dashboard & Analytics)
* **مخططات بيانية ديناميكية:** تم دمج مكتبة `Chart.js` لعرض نشاط القيود وحركة الحسابات خلال آخر 6 أشهر.
* **معالجة القيود المحاسبية:** نظام استعلام ذكي (Robust Queries) يعتمد على الأرقام الصرفة (السنة والشهر) لتجنب أخطاء ترميز التاريخ واختلاف اللغات (العربية/الإنجليزية) بين الخوادم والمتصفحات.
* **أدوات مطور مدمجة:** تم تزويد السكربتات بنظام فحص ذاتي (`console.log`) لمراقبة تدفق البيانات من قاعدة البيانات إلى المتصفح مباشرة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **الجانب البرمجي (Backend):** C# (ASP.NET Web Forms), .NET Framework.
* **قاعدة البيانات (Database):** MS SQL Server (ADO.NET للهندسة البرمجية المباشرة والاستعلامات المحددة).
* **الواجهات (Frontend):** Bootstrap 5.3 (النسخة المتوافقة مع اتجاه RTL)، HTML5, CSS3 المخصص (Custom Properties & Variables).
* **المكتبات التفاعلية:** jQuery 3.6, FontAwesome 6.4 (للأيقونات المحاسبية)، Chart.js.
